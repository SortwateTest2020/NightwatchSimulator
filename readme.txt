1. Download the chromedriver according to your local chrome version and os
    http://chromedriver.storage.googleapis.com/index.html
2. and put the driver at the root directory
3. run command: "npm install" 
4. run command: "npm run e2e"
5. to run a single test script:
   run command "./node_modules/.bin/nightwatch  case/your_script.test.js"
